# SmartBusLink 🚌
Private Bus Booking & Operator Management System

## About
A full-stack web application for managing private bus bookings, 
operators, and passengers with secure authentication.

## Features
- Passenger booking system
- Operator fleet management
- Admin dashboard
- Google OAuth authentication
- CSRF protection & secure login

## Technologies Used
- PHP, MySQL
- Bootstrap 5
- Google OAuth 2.0
- CSRF Protection
- SQL Injection Prevention

## Modules
1. Passenger Module
2. Operator Module  
3. Admin Module
